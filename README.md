# Surprise产生器

**Surprise产生器** 是一款基于 Python 开发的轻量级恶作剧程序，旨在为朋友、同事或家人带来一场意想不到的“惊喜”。

程序允许用户自由设定触发时间、自定义图片以及音效。在默认的等待时间内，程序会于背景静默运行；当到达指定时间后，将立即弹出预先设定的图片，同时播放指定的音效，营造出突如其来的惊喜效果。

## 功能特色

* 🕒 自定义延迟时间
* 🖼️ 支持自定义图片（仅 PNG、JPG/JPEG）
* 🔊 支持自定义音效或音乐
* ⚡ 背景运行，无需持续操作
* 🎭 适合愚人节、聚会、朋友之间的娱乐恶作剧

## 系统需求

* **操作系统：** Windows 10 或更新版本（Windows 10 / Windows 11）
* **目前仅支持 Windows 平台。**

## 使用场景

* 愚人节整人
* 聚会互动
* 办公室趣味娱乐
* 与朋友分享的搞笑小工具
